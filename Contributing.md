# Contributing to GRID Module

### 🐛 Reporting Bugs

I have created bug disscussion here: 

https://github.com/GHeart01/Grid-module-demo/discussions/1

**When filing a bug report, include:**
- Clear title and description
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
- Your environment (browser, OS, Node version)

### 💡 Suggesting Enhancements

I have created a feature request page here: 

https://github.com/GHeart01/Grid-module-demo/discussions/2

Please provide:
- A clear description of the feature
- Any implementation ideas you have
- Examples of similar features in other projects

### 🔧 Pull Requests

1. **Fork & Clone**
   ```bash
   git clone https://github.com/your-username/Grid-module-demo
   cd Grid-module-demo
   git remote add upstream https://github.com/GHeart01/Grid-module-demo
   ```

2. **Create a Banch
    ``` bash
    git checkout -b feature/your-feature-name
    # or
    git checkout -b fix/your-bug-fix
    ```

3. Set up Dev Environment
    ``` bash
    npm install
    npm run dev
    ```
